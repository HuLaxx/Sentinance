## üöÄ Quick Start

### 1Ô∏è‚É£ Clone & Setup

```bash
git clone https://github.com/HuLaxx/Sentinance.git
cd sentinance
cp .env.example .env
```

### 2Ô∏è‚É£ Run with Docker (Recommended)

```bash
docker-compose up -d --build
```

### 3Ô∏è‚É£ Or Run Locally

**Backend (FastAPI)**
```bash
cd apps/api
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

**Frontend (Next.js)**
```bash
cd apps/web
npm install
npm run dev
```

Visit **[http://localhost:3000/demo](http://localhost:3000/demo)** to see the magic! ‚ú®
